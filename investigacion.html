GitHub y Hosting

GitHub

GitHub es una forja para alojar proyectos utilizando Git. Es usado para crear, compartir y colaborar en código de programas. Permite almacenar tu código en un repositorio y seguir los cambios en el tiempo.

Características principales:

  • Repositorios:

    • Espacios donde se guarda el código del proyecto.

  • Git:

    • Sistema de control de versiones.

  • Colaboración:

    • Trabajo conjunto entre desarrolladores.

  • Ramas:

    • Versiones paralelas del código.

  • Pull requests:

    • Propuesta de cambios y revisiones.

  • Actions:

    • Automatiza tareas.

  • Paquetes:

    • Publicación de librerías propias.

  • Control de acceso:

    • Permisos y seguridad.

  • Issues:

    • Seguimiento de tareas o errores.

Hosting

El hosting es el servicio que proporciona almacenamiento para sitios web. Permite publicar archivos y páginas en Internet desde servidores especializados.

Características del Hosting:

  • 1. Conectividad:

    • Requiere conexión a internet para acceder.

  • 2. Almacenamiento:

    • Guarda contenido de sitios en servidores.

  • 3. Capacidad:

    • Espacio y recursos para sitios grandes o múltiples.

  • 4. Integración:

    • Incluye dominio y correo electrónico.

  • 5. Accesibilidad:

    • Protege los datos de fallos locales.
